# AI Meme & Comic Generator

A simple AI-powered web application that generates captions
and meme-style text for uploaded images.

## Why I built this
I wanted to explore how AI can be used for creative content
generation and visual storytelling.

## Features
- Upload an image
- AI-generated base caption
- Funny meme-style caption
- Caption customization
- Downloadable meme output

## Tech Stack
- Python
- Streamlit
- OpenAI API
- OpenCV
- PIL

## What this project demonstrates
- AI integration
- Image processing
- Product-focused thinking
- Creative use of technology
